Overview

In this first project, our team created an new activity tracker application which aims to give users a system that reduces fake out methods and promotes healthier behaviors and behavior changes in the users daily life. Fake out methods are any behavior that is an improper exploitation of a system and encourages the user to “cheat” through the system.

Team Members

My project partner for this project was Mason Knight.

Contribution

Working in pairs for this project, I split many of the tasks with Mason. I completed interviews, ideation (sketches), research, as well as user testing through Zoom. I focused mainly in the areas or research and ideation within this project, working with Mason to iterate on the ideas that came through my ideation.
